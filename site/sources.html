JETM Source Access

The project git repository is currently hosted at Github. Before JETM 1.3.0 it was using SVN and CVS at Sourceforge. During the transition from CVS to SVN the source structure was changed significantly, therefore SVN is missing the change history of CVS.

As usual anonymous GIT access is available.

Anonymous GIT access

The JETM git repository is located at the following URL. Feel free to fork it. 

git://github.com/frenchc/jetm.git

GIT Web Access

Since JETM is now hosted at Github you may browser the current source repository online using Github Web Access.
