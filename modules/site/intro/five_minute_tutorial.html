JETM: Five minute tutorial

This tutorial demonstrates some advanced features of JETM. It assumes you have already taken a look at advanced concepts. For a quick intro you better start with one minute tutorial.

In this tutorial we will develop a simple RMI server and measure its performance. Out of convience we will use Springframework features to exponse the RMI service, add performance monitoring and issue client requests.
(If you are new to the Springframework this tutorial may take longer than five minutes.)

Step 1: Download samples and dependencies

The archive JETM examples contains the sources and all dependencies for this tutorial. Download the file, expand it and create a new project from the content of tutorial/five-minute in your favourite IDE. Ensure that lib/jetm.jar, lib/jetm-optional.jar and all libraries in tutorial/five-minute/lib are part of your project classpath.

Alternatively you can use ant and can run this tutorial from the command line.
