JETM CVS Access

The project CVS repository is hosted at SourceForge.net. As usual anonymous and developer CVS access are available. Information about accessing this CVS repository may be found in the document titled "SourceForge.net: CVS (Version Control for Source Code)".

Anonymous CVS Access

This project's CVS repository can be checked out through anonymous (pserver) CVS with the following instruction set. When prompted for a password for anonymous, simply press the Enter key. 

cvs -d:pserver:anonymous@jetm.cvs.sourceforge.net:/cvsroot/jetm login

cvs -z3 -d:pserver:anonymous@jetm.cvs.sourceforge.net:/cvsroot/jetm co -P jetm

Developer CVS Access via SSH

Only project developers can access the CVS tree via this method. A SSH client must be installed on your client machine. Enter your site password when prompted. Developers may also make use of shared SSH keys for authentication. 

export CVS_RSH=ssh

cvs -z3 -d:ext:developername@jetm.cvs.sourceforge.net:/cvsroot/jetm co -P jetm
